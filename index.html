    What is Clean Fridge?

    Clean Fridge is an innovative app designed to take the guesswork out of meal preparation by utilizing the ingredients you have on hand.

    How does Clean Fridge work?

    Clean Fridge generates delicious recipes using the ingredients you input into the search bar.

    Why should I sign up?

    If you ever find yourself wondering what you can make with the limited ingredients you have on hand, Clean Fridge is for you!

    What if I have special dietary needs or restrictions?

    No need to worry! Clean Fridge will soon give you the ability to filter results based on different dietary options (Whole30, Paleo, vegetarian, vegan, dairy-free, gluten-free, etc.), as well as the option to create a profile and save these preferences for future searches.

    Nutrition Information:

    All Nutrition Data is displayed in a convenient graph located at the bottom of each recipe display to help with counting macros.
